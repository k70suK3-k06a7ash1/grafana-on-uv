## Project Overview

A Python project using uv as the package manager. Requires Python 3.13+.

## Commands

```bash
# Install dependencies
uv sync

# Run the application
uv run main.py

# Add a package
uv add <package-name>

# Remove a package
uv remove <package-name>
```
